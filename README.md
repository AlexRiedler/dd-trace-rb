## Testing

You can launch all tests using the following rake command:
```
  $ rake test
```

We also enforce the Ruby [community-driven style guide][1] through Rubocop. Simply launch:
```
  $ rake rubocop
```

[1]: https://github.com/bbatsov/ruby-style-guide
